=========================================================
This are explanations of files listed in /tests directory 
=========================================================

* final_result.csv is a result of a program flow without the generation. It analyses existing folders and creates a table on the basis of the data that it has picked from the existing folders.
* generate_specific_test_1.csv and shopdata_generated_specific_test_1.png are a result of generate.data.specific function with one shop being entered and two products for this shop being analysed (which have been entered by user as well)
* generate_data_test_1.in and generate_data_test_1.out are a result of generate.data function
